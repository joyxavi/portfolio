Restaurant Reservation Website

The Restaurant Reservation Website is a full-stack web application designed to allow customers to reserve tables online easily and efficiently. The website provides a user-friendly interface for customers while offering an admin dashboard for restaurant management.

Customers can submit reservation requests by selecting date, time, and number of guests. The reservation data is stored securely in the database and can be viewed and managed by administrators through a protected admin panel.

Technologies Used

  • HTML, CSS, Bootstrap for frontend design
  • PHP & CodeIgniter framework
  • MySQL database

Key Features

  • Online table reservation form
  • Admin login and authentication
  • Reservation management dashboard
  • Customer testimonial submission and approval system

What I Learned

This project helped me understand the MVC architecture using CodeIgniter, implement CRUD operations, and integrate frontend forms with backend logic. I also learned how to build admin dashboards and manage real-world user data securely.
